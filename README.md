# RobotCode2026 -  Outreach Branch - Controls Below
### left drive
- trigger: slow mode
- left button: zero heading (towards intake)
- slider: turret velocity
- translation: robot translation
-----
### right drive
- trigger: intake + shoot
- left button: toggle turret
- bottom button: outtake
- right button: hold to enable operator shooting
- rotation: robot rotation
--- 
### operator (kid driven?)
- trigger = shoot (requires driver enable)
- rotation: turret rotation +- 60 degs
